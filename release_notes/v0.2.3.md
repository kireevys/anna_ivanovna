# v0.2.3

**Дата:** 2026-03-22

Шаблоны планов для онбординга новых пользователей. Коллекция по книге «Самый богатый человек в Вавилоне» — 4 персонажа с разными жизненными ситуациями и стратегиями бюджета.

## Шаблоны планов

- 4 стратегии по мотивам книги Джорджа Клейсона:
  - **Бансир** (10/90) — «Зарабатываю, но к концу месяца ничего не остаётся»
  - **Номасир** (20/50/30) — «Стабильный доход, хочу управлять деньгами осознанно»
  - **Дабасир** (10/70/20) — «Есть кредиты или долги, хочу выбраться»
  - **Аркад** (10/50/20/20) — «Расходы под контролем, хочу приумножать»
- Принцип «Заплати себе первому» — всегда первый конверт в каждом шаблоне
- Ссылки на книгу (Литрес) и аудиоверсию (YouTube)

## Создание плана

- Экран выбора шаблона при первом запуске (нет плана → 404 → шаблоны)
- Выбор шаблона заполняет форму — пользователь настраивает и сохраняет
- Кнопка «Настрою сам» для создания с нуля
- Возврат к шаблонам без перезагрузки
- Валидация блокирует сохранение пустого плана

## API

- `GET /v1/collections` — список коллекций с шаблонами
- `POST /v1/plan` — создание плана (уже был, теперь используется фронтом)

## Прочее

- Empty state для пустой истории распределений
- Фикс пагинации: «Загрузить ещё» не показывается на последней странице
- Запрет выделения текста в Tauri (кроме инпутов)
- `pub(super)` → `pub(crate)` во frontend handler-ах
