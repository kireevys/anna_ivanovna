# v0.2.0

**Дата:** 2026-03-21

Крупный релиз: десктоп-приложение, веб-интерфейс, переезд на SQLite, async-архитектура.

## Десктоп-приложение (Tauri v2)

- Полноценное десктоп-приложение на Tauri v2
- Встроенный Axum-сервер — запустил и работаешь, ничего настраивать не нужно
- Graceful shutdown при закрытии окна
- Работает параллельно с веб-версией (разные порты)

## Веб-интерфейс (Yew)

- Полный SPA на Yew/WebAssembly
- Просмотр и редактирование плана
- CRUD строк: добавление/удаление источников дохода и расходов
- Поступление дохода с выбором источника
- История распределений с пагинацией
- Поддержка типов дохода: зарплата (gross → net с налогом) и прочие
- Абсолютные подсказки для процентных расходов
- Переключение тем (DaisyUI)

## Бэкенд

- REST API на Axum
- Переезд с файловой системы на SQLite (sqlx)
- Async на всём стеке: CoreRepo, CoreApi, Axum-хендлеры, CLI
- Слоистая архитектура: ai-core (домен) → ai-app (application) → backend (infrastructure)
- Event log для мутаций плана (план + версионирование)

## Типы дохода

- Зарплата (gross): указывается валовая сумма и ставка налога, автоматически рассчитывается сумма на руки
- Прочие доходы: фиксированная ожидаемая сумма

## Инфраструктура

- CI: проверки на PR (clippy, тесты), сборка Tauri-релизов на GitHub Actions (macOS/Windows/Linux)
- Workspace: ai-core, ai-app, backend, frontend, src-tauri
