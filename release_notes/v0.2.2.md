# v0.2.2

**Дата:** 2026-03-22

Система конфигурации приложения и онбординг при первом запуске.

## Конфигурация

- Типы конфигурации в ai-app: AppConfig, ServerConfig, DatabaseConfig (tagged enum)
- ConfigProvider trait — абстракция источника конфигурации
- CLI-провайдер: читает `{buh_home}/config.json`, поддержка `--buh-home` / `BUH_HOME`, fallback на `~/.buh`
- Tauri-провайдер: `settings.json` в app_config_dir указывает на buh_home

## Онбординг (Tauri)

- Welcome screen при первом запуске: выбор папки для данных
- AppPhase (Checking → Onboarding → Ready) управляет состоянием запуска
- Автоматическая миграция legacy-конфигурации (BUH_HOME / ~/.buh)

## Прочее

- Защита от повторного запуска бэкенда (AtomicBool guard)
- spawn_blocking для диалога выбора папки (fix tokio deadlock)
- Health check polling вместо hardcoded sleep
