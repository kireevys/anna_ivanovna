# v0.2.1

**Дата:** 2026-03-22

Переезд с CDN на локальную сборку фронтенда. Приложение больше не зависит от сети для загрузки стилей.

## Локальная сборка CSS

- Tailwind CSS и DaisyUI собираются локально через npm (devDependencies)
- Trunk pre_build hook автоматически генерирует CSS при сборке
- Удалена JS-библиотека theme-change (логика уже была реализована в Rust)

## Безопасность

- Настроена Content Security Policy (CSP) в Tauri — запрещены внешние подключения

## Прочее

- Добавлена favicon для веб-версии (из иконок Tauri, без дублирования)
- CI: добавлен Node.js setup для сборки CSS в release workflow
