# Сохранение плана — успех

## Предусловие
Приложение в состоянии редактирования плана (Editing).

## Шаги
1. Пользователь нажимает "Сохранить"
2. Бэкенд подтверждает сохранение

## Ожидаемый результат
- Приложение переходит в состояние загрузки (перезагрузка плана)
- Отправлены команды: SavePlan, LoadPlan
