# Сохранение плана — бэкенд отклонил

## Предусловие
Приложение в состоянии редактирования плана (Editing).

## Шаги
1. Пользователь нажимает "Сохранить"
2. Бэкенд отклоняет запрос (422)

## Ожидаемый результат
- Приложение возвращается в состояние редактирования с возможностью повторного сохранения
