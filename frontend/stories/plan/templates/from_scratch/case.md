# Пользователь создаёт план с нуля

## Предусловие
Приложение в состоянии выбора шаблона, шаблоны загружены (пустой список).

## Шаги
1. Пользователь нажимает "Создать с нуля"

## Ожидаемый результат
- Приложение переходит в состояние создания плана (Creating) с пустыми доходами и расходами
- Отправлена команда ScrollToTop
